## こと (Koto) Usage Patterns

| Function                 | Pattern         | Meaning                                | Example (Japanese)               | Example (English)                              |
| ------------------------ | --------------- | -------------------------------------- | -------------------------------- | ---------------------------------------------- |
| **Nominalization**       | Verb/Adj + こと | Turns verbs/adjectives into nouns      | 日本語を勉強することは楽しいです | "Studying Japanese is fun"                     |
| **Ability**              | ことができる    | "can do" / "be able to"                | 日本語を話すことができます       | "I can speak Japanese"                         |
| **Experience/Frequency** | ことがある      | "sometimes" / "have experienced"       | 寿司を食べることがあります       | "I sometimes eat sushi"                        |
| **Decision (External)**  | ことになる      | "it has been decided" / "it turns out" | 来年日本に行くことになりました   | "It's been decided I'll go to Japan next year" |
| **Decision (Personal)**  | ことにする      | "decide to" / "make it a rule to"      | 早く寝ることにします             | "I've decided to go to bed early"              |
| **Prohibition**          | ことはない      | "there's no need to" / "shouldn't"     | 心配することはありません         | "There's no need to worry"                     |
| **Indirect Speech**      | と言うこと      | "the fact that" / "saying that"        | 彼が来ないということです         | "The fact is that he won't come"               |
